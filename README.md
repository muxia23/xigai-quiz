# 习概选择题复习系统

福州大学《习近平新时代中国特色社会主义思想概论》选择题复习系统。基于 FastAPI + SQLite，单文件前端，开箱即用。

题库共 **269 题**（177 道单选 + 92 道多选），覆盖全部章节。

## ✨ 功能

- **顺序刷题**：按章节顺序逐题练习，退出后可继续
- **随机刷题**：随机抽取题目，可自选章节与数量
- **错题重练**：自动收集错题，连续答对 4 次自动移出错题集，支持导出 PDF
- **模拟考试**：20 道单选（1 分）+ 10 道多选（2 分），满分 40 分，含成绩趋势图
- **题目搜索**：按题干、选项内容或章节关键词模糊搜索，直接显示正确答案
- **统计概览**：答题进度、正确率、各章节掌握情况

## 🚀 快速开始

```bash
git clone https://github.com/muxia23/xigai-quiz.git
cd xigai-quiz
./run.sh
```

首次运行会自动创建 `.venv` 并安装依赖（fastapi / uvicorn / weasyprint），随后自动从 `26年上学期选择题汇总.json` 初始化一个**空记录**的数据库。

启动后访问 <http://127.0.0.1:8000> 即可使用。

> 也可手动安装：`pip install -r requirements.txt && python main.py`

## 📦 项目结构

```
main.py            # FastAPI 后端：题目、答题记录、错题、考试、搜索接口
database.py        # SQLite 建表与题库初始化
static/index.html  # 单文件前端
26年...json        # 题库数据源（程序据此初始化数据库）
```

## 🗄️ 关于数据库

- 运行时的 `quiz.db` 保存你个人的答题历史，**已被 `.gitignore` 排除，不会进入版本库**。
- 程序在 `quiz.db` 不存在或题目为空时会自动从 JSON 初始化一个空记录数据库，不会覆盖已有记录。
